//Task10
10. To find out which of the following are legal and which are not legal,we have to know the rules of java ;
here are the rules of java which identifiers must follow,
1.must not begin with a digit
2.should not be a Java keyword (some reserved words)
3.spaces or special characters are not allowed(except"_" and "$")
4.(A-Z),(a-z),(0-9),(_),($) are allowed

So, now lets explain if the following examples are legal or illegal in Java, 
a)hungry
int hungry;
 Result: valid
 Reason:It started with a letter and followed all rules.
b)2AB
int2AB;
Result: invalid
Reason: it starts with a number which is not allowed.
c)312.2
int312.2;
Result: invalid
Reason: it starts with a number also it contains a decimal point which is why its not valid
d)MOBILE
intMOBILE;
Result: valid
Reason: starts with a letter and follows all the rules
e)"Ans"
int"Ans";
Result: invalid
Reason: it starts with ( " ) but only (_) underscore and dollar ($) are allowed also it is a string literal
f) $30
int$30;
Result: valid
Reason:starts with (&) dollar which is valid but it is discouraged for identifiers
g)Yes/No
intYes/No;
Result: invalid
Reason:it contains a symbol (/) which is not allowed
h)student-id
intstudent-id;
Result:invalid
Reason: contains a symbol (-) hyphen which is not allowed
i)A+3
intA+3;
Result:invalid
Reason: contains a special character (+) plus which is invalid
j)`X`
int`X`;
Result: invalid
Reason: starts with a quote (`) which is a character literal not an identifier
k)return
intreturn;
Result: invalid
Reason: invalid because "return" is a JAVA keyword
